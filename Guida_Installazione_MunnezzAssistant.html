Guida ufficiale MunnezzAssistant

Installazione passo passo per Home Assistant – Versione Utente Finale

1. Requisiti minimi

  • Home Assistant versione 2024.12.0 o superiore
  • Accesso a file e cartelle sul tuo Home Assistant
  • Browser moderno
  • Non è richiesta programmazione

2. Scarica MunnezzAssistant

Vai su vesuviocode.github.io/munnezzassistant-site e scarica il pacchetto .zip

3. Scegli la tua installazione

Home Assistant OS / Supervised

  1. Vai su Impostazioni > Componenti aggiuntivi > Studio File / Samba
  2. Accedi alla cartella config/custom_components/
  3. Crea la cartella munnezzassistant
  4. Estrai qui il contenuto dello zip
  5. Riavvia Home Assistant

Home Assistant Container (Docker)

  1. Accedi al volume Docker
  2. Crea custom_components/munnezzassistant
  3. Copia i file
  4. Riavvia il container

Home Assistant Core (Python)

  1. Vai su ~/.homeassistant/custom_components/
  2. Crea munnezzassistant
  3. Inserisci i file del pacchetto
  4. Riavvia con systemctl restart home-assistant.service

4. Configura MunnezzAssistant

  1. Vai su Impostazioni > Dispositivi e servizi > Integrazioni
  2. Clicca su “+ Aggiungi integrazione”
  3. Cerca “MunnezzAssistant”
  4. Segui la procedura guidata

5. Personalizza la Dashboard

Vai su Panoramica > Modifica dashboard e aggiungi card personalizzate:

  • Tipo di rifiuto
  • Animazione (facoltativa)
  • Pulsante conferma

6. Test e verifica

  • Simula un giorno modificando il sensore o la data
  • Verifica le notifiche vocali
  • Prova il pulsante “Conferma”

7. Problemi comuni

  • Assicurati di riavviare Home Assistant
  • Controlla che ci sia manifest.json nella cartella
  • Verifica l’integrazione con Alexa/Google

8. Dove trovare supporto

  • Landing ufficiale
  • Gruppo Facebook: Home Assistant Italia
  • Segnala un problema su GitHub
