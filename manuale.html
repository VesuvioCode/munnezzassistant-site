1. Requisiti

  • Home Assistant 2024.12.0 o superiore
  • Accesso alla cartella config per aggiungere custom_components
  • Facoltativo: HACS installato per aggiunte Lovelace

2. Installazione

  1. Scarica il file ZIP di MunnezzAssistant da GitHub
  2. Estrai la cartella munnezzassistant in config/custom_components/
  3. Riavvia Home Assistant
  4. Vai su Impostazioni > Dispositivi e servizi > Aggiungi integrazione > MunnezzAssistant

3. Integrazione in Home Assistant

Una volta installata, puoi configurare:

  • Tipi di rifiuti e calendario raccolta
  • Messaggi vocali Alexa/Google
  • Utente responsabile del conferimento

4. Dashboard Lovelace

Per usare la dashboard visiva:

  1. Copia i file YAML da lovelace/ nella tua config/
  2. Vai su Impostazioni > Dashboard > 3 puntini > Importa YAML
  3. Applica anche il tema munnezzassistant_light
